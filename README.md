# CSQR-D (Curved QR Scanner)

Простое Android-приложение для сканирования QR-кодов с использованием C++ (OpenCV + ZXing) и Kotlin (CameraX UI).

---

## 📥 Скачать приложение

### 👉 [⬇ Download APK (Latest Release)](https://github.com/Akula538/CSQR-D/releases/tag/v1.0)

---

## 📱 Возможности

- Сканирование QR-кодов через камеру
- Обработка изображений на C++ (быстро и нативно)
- Поддержка ссылок (кликабельные результаты)
- Копирование текста в буфер обмена
- Возврат к камере после сканирования

---

## ⚙️ Технологии

- Kotlin (Android UI)
- C++ (NDK, OpenCV, ZXing-cpp)
- CameraX
- JNI bridge

---

## 🚀 Как использовать

1. Перейди в релиз
2. Скачай APK
3. Установи на Android устройство
4. Разреши доступ к камере
5. Сканируй QR-коды
